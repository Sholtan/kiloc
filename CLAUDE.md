# Project Rules for Claude Code

## File access policy
- You may READ any file in this repository.
- You may only PROPOSE EDITS to `.md` files.
- Never edit `.py`, `.sh`, `.yaml`, `.json`, `.toml`, or any source/config file directly.
- Always wait for explicit user confirmation before writing to any .md file.
- If asked to edit a non-.md file, refuse and explain the policy.



## Git policy
- Do not run `git push`, `git reset --hard`, or any destructive git commands.
- You may run `git status`, `git diff`, `git log`, `git add`, `git commit` only after confirmation.
